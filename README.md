# 100 Days of React

I am following the course:  
**The Ultimate React Course 2025: React, Next.js, Redux & More (Jonas Schmedtmann)**

This repository will contain:

- Daily learning folders (`/days`)
- Short notes for each day
- Practice exercises and mini-projects
- Bigger portfolio-ready apps (`/projects`)
- Weekly time summaries (Toggl Track)
